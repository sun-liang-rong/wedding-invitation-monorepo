# 创建 PC 端婚礼请帖页面计划

## 目标
在 `apps/frontend/app` 目录下新建 `pc` 目录，创建 PC 端婚礼请帖展示页面，主题色 `#FF8C42`。

## 实施步骤

### 1. 创建目录结构
```
apps/frontend/app/pc/
├── page.tsx          # PC 端婚礼请帖页面
├── layout.tsx        # PC 端布局文件
└── globals.css       # PC 端样式文件
```

### 2. 创建 PC 端布局文件 (`layout.tsx`)
- 移除移动端特有的样式限制
- 设置 PC 端合适的视口宽度
- 引入 PC 端样式文件

### 3. 创建 PC 端全局样式 (`globals.css`)
- 定义主题色变量 `--color-primary: #FF8C42`
- 设置 PC 端默认背景和字体
- 保持响应式设计

### 4. 创建 PC 端婚礼请帖页面 (`page.tsx`)
页面包含以下模块：
- **封面区域**：新人姓名（并排显示）、婚礼日期、背景封面图
- **新人信息**：新郎新娘姓名及介绍
- **婚礼信息**：时间、地点详情
- **相册展示**：照片网格展示
- **祝福语**：底部祝福文案
- **互动功能**：宾客签到按钮

使用 Tailwind CSS 进行样式设计，保持与移动端一致的设计语言。

### 5. 配置路由
确保访问 `/pc` 路径可以正常显示 PC 端页面。
